5강은 4강에서 완성한 전략 기획 패키지를 AI 도구로 재구성해 AI 포트폴리오를 완성합니다. 좌측 사이드바 단계를 따라 한 페이지씩 실습을 진행하세요.

역할 정리

  • 팀장: Notion 대시보드 관리, 제출물 취합
  • A (전략 담당자): KPI·OCR 데이터 정리, Gemini OCR 실습
  • B (콘텐츠 담당자): 이미지 리터치, Canva 조립, 문장 고도화

<보완요청> 5강 역할/흐름 개요 다이어그램 삽입

4강 산출물이 최신 상태인지 확인하고 AI 도구 접근 권한을 재점검합니다.

준비 체크리스트

  • [ ] Notion "4강 프로젝트" 페이지를 복제해 "5강 - AI 포트폴리오" 섹션 생성
  • [ ] Canva Brand Kit에 팀 색상/폰트 업로드
  • [ ] Gemini Advanced와 ChatGPT GPT-4o 계정 로그인 확인
  • [ ] 4강 Docs·Sheets·Slides 링크가 최신 버전인지 다시 연결

<보완요청> 5강 대시보드 초기 화면 캡처 삽입

팀장이 4강 산출물을 5강용 Notion 대시보드에 재정렬합니다.

  1. "자료 보관소" 데이터베이스를 생성해 Docs·Sheets·Slides·자기소개서 링크를 추가합니다.
  2. "포트폴리오 재료" 속성으로 이미지·텍스트·표를 태그화합니다.
  3. 5강용 진행 상태(불러오기/편집 중/완료) 칼럼을 추가합니다.
  4. 대시보드 상단에 제출 마감 일정과 담당자 역할 카드(간트 뷰)를 배치합니다.

성과물

Notion 대시보드 링크를 팀 채팅과 담당 교수에게 공유합니다.

<보완요청> Notion 대시보드 재정렬 화면 캡처 삽입

B 담당자가 Gemini를 사용해 이미지 톤을 직무 맞춤형으로 리터치합니다.

  1. 4강 문서·슬라이드에서 사용할 이미지 3장을 선택합니다.
  2. Gemini 이미지 편집 패널에 업로드하고 아래 프롬프트를 실행합니다.
"이 이미지를 취업 포트폴리오에 어울리는 밝은 사무실 톤으로 바꾸고, 주인공을 선명하게 강조해줘.
배경은 근현대적 회의실로 정리해줘."
  1. 배경 대체, 색감 조절, 스타일 변환(아이콘화) 등 3가지 버전을 제작합니다.
  2. 팀 피드백을 받아 최종 이미지를 PNG로 다운로드해 Canva Brand Kit에 업로드합니다.

Positive : 윤리 체크: 인물 사진 사용 시 초상권 동의 여부와 학교 AI 윤리 가이드를 준수합니다.

성과물

리터치 전/후 비교 이미지를 Notion 자료 보관소에 업로드하고, 선택 이유를 요약합니다.

<보완요청> Gemini 이미지 편집 화면 캡처 삽입

A 담당자가 Gemini OCR을 활용해 이미지 속 텍스트를 추출하고 데이터화합니다.

  1. 리터치된 이미지 또는 4강 Slides 캡처를 Gemini OCR 모드에 업로드합니다.
  2. 텍스트를 추출한 뒤 ChatGPT에 다음 요청을 전달합니다.
"추출된 텍스트를 KPI 요약 카드 형식으로 구조화해줘.
열은 항목, 핵심 문장, 활용 페이지, 후속 액션으로 구성해줘."
  1. 생성된 표를 Google Sheets에 붙여넣고 Canva 임베드용 CSV로도 저장합니다.
  2. Notion "포트폴리오 재료" 데이터베이스에 해당 표 링크와 미리보기를 등록합니다.

성과물

OCR 결과표 링크와 CSV 파일을 Drive의 "03_OCR" 폴더에 저장합니다.

<보완요청> OCR 결과 표 캡처 삽입

B 담당자가 자기소개서와 포트폴리오 설명문을 전문 어조로 다듬습니다.

  1. 4강 자기소개서와 전략 요약문을 ChatGPT에 입력합니다.
  2. 다음 프롬프트를 순차적으로 실행합니다.
"다음 텍스트를 400~450자 범위에서 전문적이고 진정성 있는 어조로 다듬어줘.
핵심 성과에는 수치를 남기고, 문장 길이는 20자 내외로 조절해줘."

"포트폴리오 인트로에 사용할 3문장 요약문을 만들어줘.
1문장은 프로젝트 임팩트, 1문장은 사용 도구, 1문장은 나의 역할을 강조해줘."
  1. 출력 결과를 Docs에 붙여넣고 팀원 피드백 후 최종본을 확정합니다.
  2. 포트폴리오에 사용할 핵심 문장 TOP3를 선정해 Notion에 하이라이트로 저장합니다.

성과물

고도화된 자기소개서 PDF와 3문장 요약을 Drive "04_Content" 폴더에 저장합니다.

<보완요청> ChatGPT 고도화 전후 비교 이미지 삽입

B 담당자가 Canva에서 포트폴리오 레이아웃을 구성하고 콘텐츠를 배치합니다.

  1. "AI Career Portfolio" 또는 유사 템플릿을 복제하고 학교 색상으로 커스터마이징합니다.
  2. Gemini 리터치 이미지를 주요 페이지에 배치하고, Brand Kit로 스타일을 통일합니다.
  3. Sheets에서 만든 KPI 표를 Canva 표 또는 임베드 기능으로 삽입합니다.
  4. ChatGPT가 생성한 요약문과 자기소개서 하이라이트를 텍스트 블록에 배치합니다.
  5. Magic Write로 페이지별 서브헤드(예: "전략 설계", "AI 적용")를 생성해 적용합니다.
  6. PDF로 내보내고 공유 링크를 "댓글 가능" 권한으로 설정합니다.

성과물

Canva 링크와 PDF 파일을 Notion "최종 산출물" 섹션 및 Drive "05_Portfolio" 폴더에 저장합니다.

<보완요청> Canva 완성본 미리보기 이미지 삽입

팀장이 모든 산출물을 Notion과 GitHub Pages 제출 구조에 맞춰 정리합니다.

  1. Notion "최종 산출물" 데이터베이스에 Canva·자기소개서·OCR 표 링크를 첨부합니다.
  2. 완료 시각을 기록하고 담당자별 회고 노트를 KPT 형식으로 작성합니다.
  3. GitHub Pages 배포 여부를 확인하고, 미리보기 URL을 Notion에 추가합니다.
  4. 제출 체크리스트(링크 공유, 파일 백업, 피드백 요청)를 모두 완료합니다.

성과물

Notion 마감 페이지 스크린샷과 미리보기 URL을 담당 교수에게 공유합니다.
