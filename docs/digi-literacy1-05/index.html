디지털 리터러시 1-05

온라인 협업과 커뮤니케이션 역량 강화를 위한 Codelab 샘플입니다.

이 세션은 온라인 협업 도구를 활용해 효율적인 커뮤니케이션을 실습하는 과정입니다. 참가자는 실시간 공동 작업과 기록 관리 방법을 익힙니다.

  • 온라인 협업 도구(예: Google Workspace) 소개
  • 소통 규칙과 역할 분담 안내
  • 실습 미션 개요 공유

다음 순서를 따라 실습 환경을 준비합니다.

1. 공유 드라이브 생성

Google Drive에서 협업용 폴더를 만들고 구성원에게 편집 권한을 부여합니다.

2. 공용 문서 템플릿 복사

아래 템플릿을 복사해 조별 기록지로 활용합니다.

https://docs.google.com/document/d/샘플템플릿ID

3. 커뮤니케이션 채널 선정

Google Chat 또는 Slack 채널을 개설하고 공지사항 메시지를 작성합니다.

조별로 협업 미션을 수행하며 다음 조건을 충족합니다.

  1. 회의 기록 문서에 회의 목적과 참여자 역할을 정리합니다.
  2. 실시간 편집 기능을 통해 회의 중 결정 사항을 업데이트합니다.
  3. 채팅 채널에 회의 요약과 후속 액션 아이템을 공유합니다.

Tip: 회의 도중 태그 기능(@)을 활용하면 담당자에게 빠르게 알릴 수 있습니다.

세션을 마무리하면서 다음 내용을 점검합니다.

  • 협업 과정의 어려움과 해결 방안을 회고합니다.
  • 공유 드라이브의 파일 구조와 접근 권한을 정돈합니다.
  • 다음 세션까지 수행할 협업 프로젝트 과제를 안내합니다.

실습 결과물을 저장소에 정리하고, 필요 시 Codelab 콘텐츠를 업데이트하여 학습자에게 공유하세요.
