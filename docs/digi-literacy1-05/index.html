4강에서 다룬 협업 워크플로를 단일 콘텐츠 담당자 관점으로 압축해, Canva와 LLM을 활용한 취업 포트폴리오 완성을 목표로 진행합니다. 전 단계에서 구축한 문서·데이터는 그대로 활용하되, 본 강의에서는 디자인과 설명력을 집중적으로 고도화합니다.

학습 목표

  • Canva의 기본/고급 편집 기능을 조합해 포트폴리오 레이아웃을 완성한다.
  • ChatGPT·Gemini·Grammarly 등 LLM을 활용해 설명 문구를 생성·개선한다.
  • AI 이미지 편집(증명사진 리터치)과 자료 분석 기능으로 설명 정확도를 높인다.

준비 체크리스트

  1. ✅ Canva 계정을 로그인하고 홈 → 프로젝트에서 4강 자료를 모은 폴더를 만든다.
  2. ✅ Chrome 기본 창에 Canva, Gemini, ChatGPT, Grammarly 탭을 고정한다.
  3. ✅ Notion 포트폴리오 재료 페이지를 열고 4강에서 정리한 링크·파일을 확인한다.
  4. ✅ 로컬 컴퓨터에 저장된 비공식 프로필 사진(정장 없이 촬영된 사진)을 준비한다.

브라우저 운영 팁

  • Canva는 Command + 1/2/3 단축키로 페이지 간 전환, ChatGPT/Gemini는 사이드 탭으로 고정해 맥락을 유지합니다.
  • 동일 파일명을 유지해 Canva-Notion 연동 시 혼선을 줄입니다.

Canva는 AI 이전에도 템플릿 필터, 브랜드 키트, 레이아웃 가이드, 버전 기록 기능으로 포트폴리오 제작에 강점을 제공했습니다. 이를 선행 학습하면 AI 기능 적용 시에도 결과물을 일관되게 유지할 수 있습니다.

  1. ✅ 템플릿 검색에서 포트폴리오 → 스타일 필터로 산업(디자인·IT·비즈니스)별 레이아웃을 비교한다.
  2. ✅ 디자인 → 브랜드에서 폰트·색상·로고를 설정하고 브랜드 제안으로 보조 팔레트를 확인한다.
  3. ✅ 캔버스 상단 파일 → 보기 → 페이지 가이드 표시로 그리드·안전 영역을 활성화한다.
  4. ✅ 파일 → 버전 기록을 열어 저장 시점 명명 방식을 YYYYMMDD_버전명으로 정한다.

성과물

  • 기본 템플릿 복제본
  • 브랜드 컬러·폰트가 적용된 스타일 프리셋
  1. ✅ 페이지 추가로 섹션을 4장(인트로/역량·프로젝트/성과지표/컨택 포인트) 구성한다.
  2. ✅ 스타일 → 모든 페이지에 적용으로 일관된 색상·폰트를 배치한다.
  3. ✅ 요소 → 선 & 모양에서 프레임과 타임라인을 활용해 프로젝트 단계를 배치한다.
  4. ✅ 앱 → Google Drive 연동 후 4강에서 작성한 Docs/Sheets 링크를 임베드 블록으로 추가한다.
