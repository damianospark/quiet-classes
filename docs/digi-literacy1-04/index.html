4강 | 팀 협업 중심의 전략 기획 프로젝트 완성

Notion·Google Workspace·ChatGPT를 활용해 디지털 협업 전략을 설계하고 실행합니다.

4강에서는 “협업과 전략 기획”을 중심으로, 디지털 협업 도구를 활용해 실제 프로젝트를 완료하는 전 과정을 경험합니다. 목표는 1) 팀 협업 구조 설계 → 2) 기획·데이터·시각화 산출 → 3) Notion 통합 관리로 이어지는 디지털 프로젝트 루프를 완성하는 것입니다.

강의 포지셔닝
  • 대상: 디지털 도구 활용 경험이 있는 대학교 3학년
  • 전제: 3강까지의 개념 학습과 도구 계정 생성이 완료되어 있다고 가정
  • 성과: 5강에서 활용할 기획 산출물 패키지 완성

모든 실습은 조별 협업을 전제로 하며, A(전략 담당자)와 B(콘텐츠 담당자)의 역할을 명확히 나누어 진행합니다.

학습 목표

  1. Notion으로 팀 단위의 시간 기반 프로젝트 계획을 수립한다.
  2. Docs·Sheets·Slides에서 협업 산출물을 제작하고 버전 이력을 관리한다.
  3. ChatGPT를 활용해 아이디어를 생성·수정·완성하는 반복형 워크플로를 설계한다.
  4. 모든 결과물과 완료 상태를 Notion에서 한 번에 관리한다.

사전 준비물

  • Google Workspace 계정: Docs, Sheets, Slides, Drive 접근 권한 필요
  • Notion 팀 워크스페이스: 조별 공유 페이지 생성 권한 확보
  • ChatGPT (GPT-4 이상) 계정: 프롬프트 실습 준비
  • 팀 역할표 템플릿 및 일정표 템플릿

<보완요청>Notion 프로젝트 템플릿 실습 화면 캡처 삽입

4강 전체 흐름은 다음과 같습니다.

단계 도구 핵심 활동 산출물 담당자
1. 프로젝트 기획 Notion 팀별 페이지 생성, 전략 주제 정의, 시간 단위 일정 설계 프로젝트 관리 페이지 팀장
2. 브레인스토밍 ChatGPT + Docs 아이디어 생성 → Docs 제안 모드 협업 1장 기획 문서 A (전략)
3. 데이터 구성 ChatGPT + Sheets KPI 초안 생성, Sheets 조건부서식·필터 설정 1장 KPI 데이터표 A (전략)
4. 시각 요약 Slides 전략 요약 슬라이드 제작, 발표자 노트 작성 1장 전략 요약 슬라이드 A (전략)
5. 자기소개서 작성 ChatGPT + Docs 프롬프트 기반 초안 → 공동 검토 및 수정 이력 기록 1장 자기소개서 문서 B (콘텐츠)
6. 결과 통합 Notion 문서 링크 첨부, 완료 시각 기록, 회고 노트 작성 프로젝트 진행 로그 팀장
  1. 팀장은 Notion에서 새 프로젝트 페이지를 만들고 타임라인 뷰를 추가합니다.
  2. 1시간 단위로 핵심 마일스톤(아이디어, 데이터, 슬라이드, 자기소개서, 통합)을 배치합니다.
  3. 각 카드에 담당자(A/B/팀장)를 지정하고, ChatGPT 프롬프트 링크를 첨부합니다.
  4. 왼쪽 사이드바에 “4강 실습 체크리스트” 데이터베이스를 추가하여 완료 여부를 추적합니다.
시간 관리 팁

1시간 단위 일정은 실제 실습 시간(총 120분)에 맞추어 6개 블록으로 나누고, 각 블록 종료 시점에 회고용 코멘트를 남기도록 안내하세요.

<보완요청>Notion 타임라인 뷰 예시 이미지 삽입

실습 절차

  1. A 담당자는 아래 프롬프트를 활용해 전략 기획 아이디어 3가지를 생성합니다.
GPT 프롬프트 예시
"대학생 취업 전략 프로젝트를 위하여, 직무 역량 강화·네트워킹·포트폴리오 세 영역에 대한 전략 아이디어를 각각 3가지씩 제안해줘. 각 아이디어는 실행 단계, 요구 도구, 기대 성과를 포함해줘."
  1. 생성된 아이디어를 Google Docs에 붙여넣고, 제안 모드를 활성화해 팀원 피드백을 유도합니다.
  2. B 담당자와 팀장은 댓글 기능으로 타당성 질문을 남기고, 해결된 댓글은 즉시 닫습니다.

Docs 버전 이력을 통해 초기 초안과 최종본의 변화를 비교하며 협업 흔적을 시각화합니다.

<보완요청>Docs 제안 모드 캡처 삽입

ChatGPT → Sheets 전환

  1. ChatGPT에 KPI 템플릿 생성 프롬프트를 제공합니다.
"취업 전략 프로젝트 KPI를 정량·정성 지표로 나눠서 표로 정리해줘.
열은 목표, 지표명, 측정 방법, 목표치, 주기, 책임자를 포함해줘."
  1. 생성된 표를 Sheets에 붙여넣고, 조건부서식으로 목표 대비 현재 상태를 가시화합니다.
  2. 필터 뷰를 생성해 담당자별 KPI를 빠르게 추출합니다.
  3. 실시간 차트를 추가해 KPI 달성률을 시각화하고, Docs 기획서에 링크합니다.
협업 기능 체크
  • 범례 편집 시 공동 편집자 충돌 여부 확인
  • Sheet 댓글을 통해 데이터 근거를 문서화

<보완요청>Sheets 조건부서식 예시 스크린샷 삽입
  1. Sheets에서 생성한 차트를 Slides에 임포트하여 실시간 데이터 연동을 설정합니다.
  2. 하나의 슬라이드에 핵심 메시지, KPI 차트, 실행 타임라인을 구성합니다.
  3. 발표자 노트에 “다음 단계 제안” 3가지를 ChatGPT로 생성해 추가합니다.
  4. 버전 이력에서 “실습 완료”라고 이름 붙인 체크포인트를 남겨 추후 비교합니다.

Slides 공유 권한을 “링크가 있는 모든 사용자(댓글 가능)”로 설정해 수업 후 피드백 수렴 경로를 마련합니다.

<보완요청>Slides 최종본 미리보기 이미지 삽입

B 담당자 실습 안내

  1. 다음 프롬프트를 활용해 자기소개서 초안을 작성합니다.
"전략 기획 프로젝트를 수행한 경험을 토대로, 데이터 기반 의사결정 역량을 강조하는 자기소개서 500자 초안을 작성해줘.
도입-경험-성과-다짐 구조로 구성하고, 구체적인 수치를 포함해줘."
  1. Docs에 초안을 붙여넣고, 팀원은 제안 모드와 댓글을 활용해 수정 의견을 남깁니다.
  2. 수정 이력에서 핵심 개선 포인트(예: 수치 보강, 역할 명시)를 태그로 정리합니다.
평가 기준
  • 문장 명료성: 5점
  • 성과 지표 사용 여부: 5점
  • 협업 경험 강조: 5점

<보완요청>자기소개서 제안 모드 캡처 삽입
  1. 완성된 Docs, Sheets, Slides 링크를 Notion 프로젝트 페이지에 삽입합니다.
  2. 각 카드에 완료 시각 속성을 추가하고, 체크리스트 완료 여부를 기록합니다.
  3. 팀 회고 섹션을 만들고 “Keep, Problem, Try” 형식으로 간단 회고를 진행합니다.
  4. 5강에서 활용할 항목(포트폴리오 재료)으로 태그를 지정합니다.

<보완요청>Notion 프로젝트 완료 페이지 캡처 삽입
  • Docs·Sheets·Slides에서 댓글과 제안이 모두 해소되었는지 확인한다.
  • Notion 타임라인의 모든 카드가 “완료” 상태인지 체크한다.
  • ChatGPT 대화 링크를 Notion에 남겨 추후 참고할 수 있게 한다.

체크가 완료되면 5강에서 활용할 산출물 패키지를 Google Drive 폴더에 정리해 공유합니다.

4강의 모든 산출물은 5강에서 Canva·Gemini를 활용한 포트폴리오 제작의 재료가 됩니다. 아래 항목을 꼭 완료한 뒤 5강을 준비하세요.

  • 기획서, KPI 표, 전략 슬라이드, 자기소개서 문서를 하나의 폴더로 정리
  • Notion 페이지에 “5강 준비” 체크리스트 추가
  • 선호하는 포트폴리오 템플릿(Canva) 후보 2개 조사

5강에서는 AI 기반 시각화와 콘텐츠 고도화를 통해, 본 강에서 만든 전략 산출물을 포트폴리오로 전환합니다.
