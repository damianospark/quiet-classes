# 5강 | Canva + Gemini + ChatGPT로 AI 포트폴리오 완성

## 오리엔테이션
Duration: 3

5강은 4강에서 완성한 전략 기획 패키지를 AI 도구로 재구성해 **AI 포트폴리오**를 완성합니다. 좌측 사이드바 단계를 따라 한 페이지씩 실습을 진행하세요.

### 역할 정리

- **팀장**: Notion 대시보드 관리, 제출물 취합
- **A (전략 담당자)**: KPI·OCR 데이터 정리, Gemini OCR 실습
- **B (콘텐츠 담당자)**: 이미지 리터치, Canva 조립, 문장 고도화

> **실습 운영 방식**: 4강과 동일하게 한 명이 여러 역할을 담당합니다. 기본 Chrome 창은 팀장, 시크릿 창 1은 A 담당자, 시크릿 창 2는 B 담당자로 고정해 작업하며, 각 창에서 생성한 문서는 **공유 버튼 → 링크 복사**로 서로 전달하고 Notion에 기록합니다.

> **<보완요청>** 5강 역할/흐름 개요 다이어그램 삽입

## 실습 환경 리셋
Duration: 5

4강에서 구성한 세 개의 브라우저 창을 유지하면서 5강 실습에 맞게 자료를 정리합니다.

### 준비 체크리스트
- [ ] 팀장 창: Notion `프로젝트 보드` 페이지를 복제해 `5강 - AI 포트폴리오` 섹션을 만들고 공유 링크를 복사해 둔다.
- [ ] 팀장 창: `/데이터베이스 - 테이블`로 `자료 보관소`와 `포트폴리오 재료` 테이블을 인라인으로 배치한다.
- [ ] 시크릿 창 1(A): Google Docs 기획 문서, Google Sheets KPI·OCR 시트 공유 링크를 확인하고 Notion 테이블에 기록한다.
- [ ] 시크릿 창 2(B): Google Docs 자기소개서, Google Slides 포트폴리오 슬라이드 공유 링크를 확인한다.
- [ ] Canva Brand Kit와 Gemini Advanced, ChatGPT GPT-4o 계정이 각각 로그인되어 있는지 점검한다.

### 브라우저 운영 팁
- 창 제목을 `팀장/전략(A)/콘텐츠(B)`로 바꿔 혼선을 줄이고, 링크 전달은 모두 Notion 테이블을 통해 관리한다.
- 공유 링크는 `공유 → 링크 있는 모든 사용자 → 댓글 가능`으로 통일해 두 창에서 동시에 접근 가능하게 한다.

> **<보완요청>** 5강 대시보드 초기 화면 캡처 삽입

## Notion 프로젝트 불러오기
Duration: 8

팀장이 4강 산출물을 5강용 Notion 대시보드에 재정렬합니다.

1. `/데이터베이스 - 테이블(인라인)`으로 `자료 보관소`를 만들고 행마다 Docs·Sheets·Slides·자기소개서를 등록합니다. `문서 링크` 속성에 각 공유 URL을 붙여넣고 `역할` 속성으로 작성 창(A/B/팀장)을 기록합니다.
2. `/속성 추가 → 다중 선택`으로 `포트폴리오 재료`를 만들고 이미지·텍스트·표 등 유형 태그를 지정합니다.
3. `+ 속성` 버튼으로 `진행 상태`(준비/편집/검토/완료), `마감일`, `비고` 속성을 추가해 진행 상황을 한눈에 볼 수 있게 합니다.
4. 대시보드 상단에 `/연결된 데이터베이스`를 이용해 `프로젝트 보드`를 불러온 뒤 `보드(View) → 그룹화: 역할`로 바꿔 역할별 작업을 확인합니다.

### 성과물
정리된 Notion 대시보드 공유 링크와 `자료 보관소` 테이블 스크린샷을 팀 채팅과 담당 교수에게 공유합니다.

> **<보완요청>** Notion 대시보드 재정렬 화면 캡처 삽입

## Gemini 이미지 리터치
Duration: 15

B 담당자가 Gemini를 사용해 이미지 톤을 직무 맞춤형으로 리터치합니다.

1. 4강 문서·슬라이드에서 사용할 이미지 3장을 선택합니다.
2. Gemini 이미지 편집 패널에 업로드하고 아래 프롬프트를 실행합니다.

```
"이 이미지를 취업 포트폴리오에 어울리는 밝은 사무실 톤으로 바꾸고, 주인공을 선명하게 강조해줘.
배경은 근현대적 회의실로 정리해줘."
```

3. 배경 대체, 색감 조절, 스타일 변환(아이콘화) 등 3가지 버전을 제작합니다.
4. 팀 피드백을 받아 최종 이미지를 PNG로 다운로드해 Canva Brand Kit에 업로드합니다.

Positive
: **윤리 체크**: 인물 사진 사용 시 초상권 동의 여부와 학교 AI 윤리 가이드를 준수합니다.

### 성과물
리터치 전/후 비교 이미지를 Notion 자료 보관소에 업로드하고, 선택 이유를 요약합니다.

> **<보완요청>** Gemini 이미지 편집 화면 캡처 삽입

## Gemini OCR → 데이터 정리
Duration: 12

A 담당자가 Gemini OCR을 활용해 이미지 속 텍스트를 추출하고 데이터화합니다.

1. 리터치된 이미지 또는 4강 Slides 캡처를 Gemini의 **Upload image → OCR** 모드에 업로드합니다.
2. 추출된 텍스트를 ChatGPT에 전달하며 아래 프롬프트를 사용해 TSV 형식으로 재구성합니다.

```
"다음 텍스트를 KPI 요약 카드 표로 정리해줘. 열은 항목	핵심 문장	활용 페이지	후속 액션으로 구성하고, TSV(탭 구분) 형식으로 출력해줘."
```

3. 시크릿 창 1(A)에서 Google Sheets 새 문서를 열고 TSV를 붙여넣은 뒤, **파일 → 다운로드 → 탭으로 구분된 값(.tsv)**로 저장해 Canva 업로드용 파일을 확보합니다.
4. `자료 보관소` 테이블에 `OCR 표` 행을 추가하고 `문서 링크`에 Sheets 공유 URL을, `첨부` 속성에 TSV 파일을 업로드합니다.

### 성과물
TSV 파일과 Sheets 공유 링크를 Notion `자료 보관소` 테이블에 등록합니다.

> **<보완요청>** OCR 결과 표 캡처 삽입

## ChatGPT 문장 고도화
Duration: 15

B 담당자가 자기소개서와 포트폴리오 설명문을 전문 어조로 다듬습니다.

1. 4강 자기소개서와 전략 요약문을 ChatGPT에 입력합니다.
2. 다음 프롬프트를 순차적으로 실행합니다.

```
"다음 텍스트를 400~450자 범위에서 전문적이고 진정성 있는 어조로 다듬어줘.
핵심 성과에는 수치를 남기고, 문장 길이는 20자 내외로 조절해줘."
```

```
"포트폴리오 인트로에 사용할 3문장 요약문을 만들어줘.
1문장은 프로젝트 임팩트, 1문장은 사용 도구, 1문장은 나의 역할을 강조해줘."
```

3. 출력 결과를 Docs에 붙여넣은 뒤, **공유 → 링크 있는 모든 사용자 → 댓글 가능**으로 설정하고 팀원 피드백을 반영해 최종본을 확정합니다.
4. **파일 → 다운로드 → PDF 문서(.pdf)**로 저장하고, 핵심 문장 TOP3는 Notion `포트폴리오 재료` 테이블의 `요약` 속성에 기록합니다.

### 성과물
Docs 공유 링크와 PDF 파일을 Notion `포트폴리오 재료` 테이블에 첨부하고, 핵심 문장 TOP3를 `요약` 속성에 남깁니다.

> **<보완요청>** ChatGPT 고도화 전후 비교 이미지 삽입

## Canva 포트폴리오 조립
Duration: 25

B 담당자가 Canva에서 포트폴리오 레이아웃을 구성하고 콘텐츠를 배치합니다.

1. Canva에서 `디자인 만들기 → 프레젠테이션 (1920×1080)`을 선택한 뒤 왼쪽 `템플릿` 패널에서 "AI Career Portfolio"를 검색해 복제합니다.
2. 상단 `디자인 → 브랜드 키트`에서 프로젝트 색상과 폰트를 적용하고, `업로드 → 파일 업로드`로 Gemini 리터치 이미지를 불러온 뒤 드래그해 배치합니다.
3. KPI 표를 삽입하려면 왼쪽 `앱 → Google Drive → 연결` 후 Sheets를 선택하거나, `삽입 → 표 → 4×4`로 표를 만든 뒤 TSV 파일을 붙여넣습니다.
4. ChatGPT가 제공한 요약문은 `텍스트 → 본문 추가`로 텍스트 박스를 만들고 붙여넣습니다. Magic Write를 활용하려면 캔버스에서 `/magic`을 입력해 페이지별 서브헤드를 생성합니다.
5. 페이지 구성이 끝나면 우측 상단 `공유 → 링크 있는 모든 사용자 → 댓글 가능`으로 설정하고, **다운로드 → 파일 형식: PDF 표준**을 선택해 포트폴리오 PDF를 저장합니다.

### 성과물
Canva 공유 링크와 PDF 파일을 Notion `최종 산출물` 섹션에 `/파일` 블록으로 업로드합니다.

> **<보완요청>** Canva 완성본 미리보기 이미지 삽입

## 결과 통합 & 마감
Duration: 10

팀장이 모든 산출물을 Notion과 GitHub Pages 제출 구조에 맞춰 정리합니다.

1. Notion `최종 산출물` 데이터베이스에서 `+ 새 행`을 눌러 Canva, 자기소개서, OCR 표 등을 등록하고 `문서 링크`와 `첨부` 속성에 공유 URL과 PDF/TSV 파일을 붙여넣습니다.
2. `프로젝트 보드`의 각 행을 열어 `상태`를 `완료`, `마감일`을 실제 완료 날짜로 업데이트하고 `검토 메모`에 주요 수정 사항을 정리합니다.
3. `/데이터베이스 - 테이블`로 만든 `회고(Log)`에 팀원별 Keep/Problem/Try를 작성하고, GitHub Pages 미리보기 URL은 `/북마크` 블록으로 추가합니다.
4. `/할 일` 블록으로 제출 체크리스트를 만들고 `링크 공유`, `PDF 업로드`, `담당 교수 보고`, `GitHub Pages 확인` 항목을 모두 체크합니다.

### 성과물
최종 정리된 Notion 공유 링크와 체크리스트 완료 스크린샷을 담당 교수에게 전달합니다.

> **<보완요청>** Notion 마감 체크리스트 캡처 삽입

## 품질 점검
Duration: 6

최종 제출 전 반드시 확인해야 할 항목입니다.

- [ ] 포트폴리오 PDF·Canva 링크·자기소개서·OCR 표가 Notion에서 정상적으로 열리고, 모든 공유 권한이 `댓글 가능`인지 확인한다.
- [ ] AI 이미지 사용에 대한 출처/동의 문구가 포트폴리오에 포함되어 있는지 검토한다.
- [ ] Notion `최종 산출물` 데이터베이스와 `프로젝트 보드`의 `상태`가 모두 완료로 표시되는지 확인한다.
- [ ] GitHub Pages 미리보기 URL을 별도 창에서 열어 최신 버전이 반영되었는지 확인한다.

누락이 발견되면 해당 단계로 돌아가 수정 후 다시 체크하세요.

## 확장 미션
Duration: 4

- 포트폴리오 영문 페이지를 생성해 글로벌 지원 버전으로 확장
- Gemini로 생성한 이미지를 Canva 동영상 템플릿에 적용해 모션 그래픽 제작
- ChatGPT로 인터뷰 예상 질문/답변 카드 작성 후 Notion에 보완

확장 미션 결과는 희망하는 학생만 제출하며, 추가 피드백을 제공할 수 있습니다.
