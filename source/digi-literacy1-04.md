# 4강 | 팀 협업 중심의 전략 기획 프로젝트 완성

## 0. 오리엔테이션
Duration: 3

4강은 "협업과 전략 기획"을 완성하기 위한 프로젝트 기반 수업입니다. 좌측 사이드바 단계에 따라 이동하며 각 실습을 순차적으로 진행하세요.

### 역할 정리

- **A (전략 담당자)**: 기획 아이데이션·KPI·Slides 총괄
- **B (콘텐츠 담당자)**: 자기소개서 작성, 문장 교정, 공유 메시지 관리
- **팀장**: Notion 일정·산출물 통합, 회고 수집 및 보고

> **<보완요청>** 역할 분담 다이어그램 이미지 삽입

## 1. 실습 환경 세팅
Duration: 5

조별 협업 환경을 동일하게 맞추어 권한과 템플릿을 정렬합니다.

### 준비 체크리스트

- [ ] Notion 워크스페이스에 "4강 전략 프로젝트" 페이지 생성
- [ ] Google Drive 공유 폴더 생성 후 Docs·Sheets·Slides 편집 권한 부여
- [ ] ChatGPT 대화 아카이브용 "프롬프트 기록" 섹션을 Notion에 추가
- [ ] 역할 배정표와 연락처를 Notion 하위 페이지에 정리

### 참고 자료

- [Notion 페이지 생성 가이드](https://www.notion.so/help/guides/creating-a-notion-page)
- [Google Drive 공유 폴더 설정](https://workspace.google.com/learning-center/products/drive/)

### 성과물
Notion 페이지 링크와 Drive 폴더 링크를 팀 채팅방에 게시합니다.

> **<보완요청>** 실습 환경 확인 체크리스트 캡처 삽입

## 2. Notion 타임라인 설계
Duration: 12

팀장이 프로젝트 일정을 타임라인으로 구성하고 단계별 완료 조건을 명시합니다.

1. Notion 타임라인 데이터베이스를 만들고 속성(담당자, 상태, 시작/종료, 산출물 링크)을 추가합니다.
2. 6개 마일스톤(아이데이션, KPI, Slides, 자기소개서, 통합, 리뷰)을 1시간 단위로 배치합니다.
3. 각 카드에 완료 조건을 체크박스로 기록하고 예상 리스크/대응책 메모를 추가합니다.
4. 페이지 상단에 "오늘의 목표" 콜아웃 블록을 만들고 실습 시작 시 업데이트합니다.

### 성과물
완성된 타임라인 스크린샷을 캡처해 Notion 페이지 표지 갤러리에 첨부합니다.

> **<보완요청>** Notion 타임라인 구성 예시 이미지 삽입

## 3. ChatGPT 브레인스토밍 & Docs
Duration: 15

A 담당자가 ChatGPT와 Google Docs로 전략 기획 아이디어를 생성하고 협업 피드백을 반영합니다.

### 실습 절차

1. ChatGPT에 아래 프롬프트를 입력합니다.

```
"대학생 취업 전략 프로젝트를 위해 직무 역량·네트워킹·포트폴리오 강화를 각각 3단계 실행 계획으로 제안해줘.
각 단계에 필요 도구, 예상 산출물, KPI 후보를 포함해줘."
```

2. 출력 결과를 Docs 템플릿에 붙여넣고 제안 모드로 전환합니다.
3. B 담당자와 팀장은 댓글로 보완 의견을 남기고 해결된 항목은 "해결됨" 처리합니다.
4. 버전 이력에 "브레인스토밍 확정본"을 저장한 뒤 링크를 Notion 타임라인 카드에 첨부합니다.

### 성과물
Docs 최종 링크와 핵심 의사결정 요약 3줄을 Notion 회의록 섹션에 기록합니다.

> **<보완요청>** Docs 제안 모드 전/후 비교 이미지 삽입

## 4. KPI 데이터 시각화
Duration: 15

A 담당자가 KPI를 정리하고 Google Sheets에서 시각화 대시보드를 구축합니다.

1. ChatGPT에 KPI 구성 프롬프트를 전달해 표 초안을 생성합니다.

```
"다음 전략 기획을 KPI로 정리해줘. 열은 목표, 지표명, 측정 방법, 기준값, 목표값, 측정 주기, 책임자로 구성해줘."
```

2. 생성된 표를 Sheets에 붙여넣고 조건부서식으로 진행 상태를 색상으로 표시합니다.
3. 필터 보기(담당자별, 지표 유형별)를 만들어 협업 편의성을 높입니다.
4. 막대 그래프/도넛 그래프를 생성해 Slides로 임포트 가능한 차트를 준비합니다.

### 실습 팁
- "현재 진행률" 열을 추가해 오늘 기준 self-report를 기록하세요.
- Sheets 댓글에 KPI 근거 데이터를 링크해 검증 가능성을 확보하세요.

### 성과물
Sheets 링크와 차트 URL을 Docs·Notion 타임라인 카드에 삽입합니다.

> **<보완요청>** 조건부서식 적용된 KPI 시트 이미지 삽입

## 5. 전략 슬라이드 요약
Duration: 12

A 담당자가 Slides 한 장으로 전략 스토리를 시각화합니다.

1. Sheets 차트를 Slides로 임포트하고 실시간 연동을 확인합니다.
2. 레이아웃을 3구역(전략 요약, KPI 그래프, 실행 타임라인)으로 구성합니다.
3. ChatGPT에 "30초 엘리베이터 피치"를 요청해 발표자 노트에 삽입합니다.
4. 버전 이력에 "전략 요약 v1"을 저장하고 공유 권한을 "댓글 가능"으로 설정합니다.

### 성과물
Slides 링크와 PDF 버전을 Drive 폴더에 저장한 뒤 Notion 회고 섹션에 첨부합니다.

> **<보완요청>** Slides 완성본 썸네일 삽입

## 6. 자기소개서 협업 편집
Duration: 18

B 담당자가 프로젝트 경험 기반 자기소개서를 작성하고 팀 피드백을 반영합니다.

1. 다음 프롬프트로 초안을 생성합니다.

```
"전략 기획 프로젝트에서 데이터 기반 의사결정을 이끈 경험을 500자 이내 자기소개서로 작성해줘.
도입-과제-행동-성과-다짐 구조를 따르고 KPI 수치를 포함해줘."
```

2. Docs에 붙여넣은 뒤 제안 모드로 전환합니다.
3. A 담당자와 팀장은 표현 명료성·정량 근거·협업 역할 관점에서 검토합니다.
4. 최종본을 PDF로 내보내 "자기소개서_v1.pdf"로 Drive에 저장합니다.

### 검토 포인트
- [ ] 핵심 문장마다 구체적 수치가 포함되어 있는가?
- [ ] 문장 길이가 20~25자 범위 안에서 읽기 쉬운가?
- [ ] 협업 경험과 개인 기여가 명확히 구분되는가?

### 성과물
Docs 링크·PDF 파일·수정 요약을 Notion 회고 섹션에 업로드합니다.

> **<보완요청>** Docs 제안 모드 캡처 삽입

## 7. Notion 통합 & 회고
Duration: 10

팀장이 모든 산출물을 Notion 프로젝트 페이지에 연결하고 회고 로그를 정리합니다.

1. 타임라인 각 카드에 Docs, Sheets, Slides, PDF 링크를 첨부합니다.
2. 완료 시각·피드백 필요 여부를 속성으로 업데이트합니다.
3. 하단에 "회고(Log)" 데이터베이스를 만들고 KPT 항목을 작성합니다.
4. 5강 준비 체크리스트(포트폴리오용 이미지, 텍스트, KPI 표)를 추가합니다.

### 성과물
Notion 공개 링크와 회고 요약 3줄을 팀 채팅과 담당 교수에게 공유합니다.

> **<보완요청>** Notion 회고 보드 캡처 삽입

## 8. 협업 품질 점검
Duration: 6

팀 전체가 품질 점검 리스트를 확인합니다.

- [ ] Docs·Sheets·Slides에 미해결 댓글이 없는지 확인합니다.
- [ ] Drive 폴더 구조가 "01_Docs / 02_Sheets / 03_Slides / 04_Resume"로 정리되었는지 검토합니다.
- [ ] ChatGPT 대화 URL을 Notion "참고 자료" 섹션에 기록합니다.
- [ ] Notion 타임라인 모든 카드가 "완료" 상태인지 확인합니다.

검증 후 Drive 폴더를 학교 계정과 개인 계정에 각각 백업합니다.

## 9. 확장 미션
Duration: 4

- 직무별 맞춤 전략 요약을 ChatGPT로 생성해 Notion 자료실에 추가
- Slides를 GIF 애니메이션으로 변환해 포트폴리오 삽입 가능성 테스트
- 다른 팀과 산출물을 교차 피드백하는 패널 토론 주제 선정

확장 미션은 선택 사항이며, 수행 결과를 5강 준비 자료에 첨부하면 추가 피드백을 받을 수 있습니다.
